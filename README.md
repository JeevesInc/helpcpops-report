# helpcpops-report (sem autenticação)

Weekly operational report for #help-paymentops (Cards & Payments Ops).
Versão pública, sem tela de login — abre direto no relatório.

Hosted with GitHub Pages: https://jeevesinc.github.io/helpcpops-report/

## Structure
- `index.html` — relatório atual (abre direto, sem login)
- `reports/` — relatórios arquivados por semana (YYYY-MM-DD.html)
- `.nojekyll` — serve os arquivos como estão
